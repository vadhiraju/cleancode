Hotel Automation Controller Application:


1. Inputs to the controller

For constructing Hotel floors and corridors:
--------------------------------------------
a.Number of Floors          - any integer value, application uses 2.
b.Number of Main Corridors  - any integer value, application uses 1.
c.Number of Sub Corridors   - any integer value, application uses 2.

2. Set of commands for the next inputs:

Note: commands should be in the same format.

a.Default State
b.Floor 1,Sub corridor 2
c.No movement 


3. Run the Application from the com.hotelautomationcontroller.main package.
4. Application can be tested using test classes under respective test package.
